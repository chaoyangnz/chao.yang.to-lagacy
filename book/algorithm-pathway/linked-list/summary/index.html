Linked list is a quite common data structure. Although it is easy for insertion and deletion, but sometimes it can be very tricky to use, because for a current node, its context is just its next node and its previous one if any.

dummy head

I place dummy head as the first point, because I suppose it is quite convenient when handling some problems.

dummy head is as its name: it has no extra purpose, just for head.

It is useful when we construct a linked list from scratch or the real head node is possibly changed - delete, replace etc.

commonly-use operations

reverse

Always store the reversed linked list, say ‘newHead`

  • Initial state: newHead = null, means empty list
  • Each iteration:
node.next = newHead;
newHead = node;
